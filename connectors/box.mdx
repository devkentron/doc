Capture critical data seamlessly from Box. Analyze communications and unlock powerful insights with on-demand data collection.

Kentron's Unified Injection Engine supports collecting the following data:

- File name
- Created/updated timestamps
- Creator/updater names
- File path
- Version number
- Collaborators
- Sharing status

## **Connector Overview**

The Box connector enables authorized, audit-ready access to Box Enterprise content for discovery, compliance, and governance. The integration supports full archiving, in-place preservation, identity mapping, and comprehensive metadata capture across all supported file types and collaboration activities.
